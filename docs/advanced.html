üöÄ Quick Start for Technical Users

üí° Preferred Method: Pull Request Submission

Technical users can submit data directly via Pull Requests to our private data collection repository. This method provides better version control and allows for technical discussion.

üìã Pull Request Workflow

Request Access to Private Repository

Contact the project maintainers by creating an issue in the public repository requesting access to jonobg/battery-data-collection. Include your GitHub username and brief background.

Request Access

Clone the Data Collection Repository

git clone https://github.com/jonobg/battery-data-collection.git cd battery-data-collection

Create Submission Branch

git checkout -b submission/milwaukee-m18-5ah-$(date +%Y%m%d)

Add Your Data Files

Create a new directory under submissions/ with the following structure:

submissions/ ‚îî‚îÄ‚îÄ YYYY-MM-DD_manufacturer_model/ ‚îú‚îÄ‚îÄ metadata.json ‚îú‚îÄ‚îÄ logs/ ‚îÇ ‚îú‚îÄ‚îÄ diagnostic_log.txt ‚îÇ ‚îú‚îÄ‚îÄ protocol_capture.pcap ‚îÇ ‚îî‚îÄ‚îÄ register_dump.hex ‚îî‚îÄ‚îÄ README.md

Create Metadata File

Use this JSON template for metadata.json:

{ "submission": { "id": "unique-submission-id", "timestamp": "2025-09-15T10:00:00Z", "submitter": { "github_username": "your-username", "email": "optional@example.com" } }, "battery": { "manufacturer": "milwaukee", "model": "M18", "capacity_ah": 5.0, "chemistry": "li-ion", "part_number": "48-11-1850", "manufacturing_date": "2024-03", "condition": "used" }, "device": { "type": "drill", "model": "M18 FUEL", "usage_context": "diagnostic_test" }, "capture": { "method": "uart_sniffer", "hardware": "Arduino Uno + Logic Level Converter", "software": "custom_logger_v2.py", "sampling_rate": "115200_baud", "duration_minutes": 15 }, "files": { "diagnostic_log.txt": { "type": "ascii_log", "description": "Battery register dump during charging cycle" }, "protocol_capture.pcap": { "type": "packet_capture", "description": "Raw UART communication between battery and charger" }, "register_dump.hex": { "type": "hex_dump", "description": "Complete memory dump from battery MCU" } } }

Create Pull Request

git add . git commit -m "Add Milwaukee M18 5.0Ah diagnostic data" git push origin submission/milwaukee-m18-5ah-$(date +%Y%m%d)

Then create a Pull Request with detailed description of your submission.

Arduino-Based UART Sniffer

// Basic Arduino sketch for battery communication logging #include <SoftwareSerial.h> SoftwareSerial batterySerial(2, 3); // RX, TX pins void setup() { Serial.begin(115200); batterySerial.begin(9600); // Adjust based on battery protocol Serial.println("Battery Communication Logger Started"); } void loop() { if (batterySerial.available()) { String data = batterySerial.readString(); Serial.print(millis()); Serial.print(","); Serial.println(data); } }

  1. Capture - Use logic analyzer or UART sniffer
  2. Decode - Apply manufacturer-specific protocol parsers
